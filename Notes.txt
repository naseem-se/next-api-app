# Next.js API Project

## Description
This is a Next.js application that demonstrates API integration. The project consists of a main page with project details and a products page that fetches data from an external API.

## Features
- Main page with project information
- Products page with API integration
- Navigation between pages
- Dynamic data fetching

## Technologies Used
- Next.js
- React
- Tailwind CSS
- API Integration

## Project Structure
- Main Page (index.js)
    - Project details
    - Link to Products page
- Products Page
    - API integration
    - Product listing

## Installation
1. Clone the repository:
        ```
        git clone <repository-url>
        ```
2. Navigate to the project directory:
        ```
        cd api-app
        ```
3. Install dependencies:
        ```
        npm install
        ```
4. Run the application:
        ```
        npm run dev
        ```
